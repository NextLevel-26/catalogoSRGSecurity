# SRG Seguridad - Guía de Imágenes

## Imágenes Actuales
Tienes 7 imágenes reales de productos.

## Cómo Agregar Imágenes Manualmente

### Paso 1: Descarga la imagen
Busca la imagen del producto en:
- Google Imágenes
- Página oficial de la marca
- Tu distribuidor

### Paso 2: Renombra el archivo
El nombre del archivo debe coincidir con el código del producto (sin mayúsculas, guiones en lugar de espacios).

#### Ejemplos:
| Producto | Nombre del archivo |
|----------|-------------------|
| XVR 1080 4CH | `xvr-4ch.jpg` |
| DH-IPC-HFW1230S1P-0280B-S6 | `dh-ipc-hfw1230s1p-0280b-s6.jpg` |
| NVR1104HS-S3-H | `nvr-1104hs-s3-h.jpg` |
| DS-7204HGHI-M1 | `ds-7204hghi-m1.jpg` |
| BULLET B1A21P | `bullet-b1a21p.jpg` |

### Paso 3: Guarda en la carpeta
Guarda la imagen en la carpeta `images/` con el nombre correcto.

## Productos Sin Imagen
Los productos sin imagen mostrarán un icono SVG elegante según su tipo:
- Bullet → Icono de cámara alargada
- Domo → Icono de cámara redonda
- XVR/NVR → Icono de grabador
- Switch → Icono de switch de red
- Alarmas → Icono de campana/alarma

## Nota
Si no tienes imagen de un producto específico, puedes dejarlo sin imagen. El icono SVG servirá como referencia visual.
